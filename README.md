
# E-ID Card Generator

## Overview
The E-ID Card Generator is a web-based application that allows students to generate electronic ID cards by entering basic details. The application provides a real-time preview, generates a barcode for identification, and manages validity and history using browser storage.

This project demonstrates frontend development, client-side state management, and deployment using modern web technologies.

---

## Features
- Dynamic student E-ID card generation
- Real-time preview while entering details
- Barcode generation for student identification
- Dark and light theme toggle
